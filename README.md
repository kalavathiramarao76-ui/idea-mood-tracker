# Mood Tracker

Log your daily mood with emojis, visualize trends over time with charts, and get wellness tips based on your patterns.

## Features
- Daily mood logging with emoji selection and notes
- Trend visualization with interactive charts (7/30/90 day views)
- Personalized wellness tips based on mood patterns
- CSV data export
- All data stored locally in your browser (no account needed)

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (charts)
- localStorage (persistence)

## Getting Started
```bash
npm install
npm run dev
```
